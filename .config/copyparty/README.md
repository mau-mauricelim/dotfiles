# Copyparty and cloudflare

## Windows Quick install

```cmd
> curl -sSf https://raw.githubusercontent.com/mau-mauricelim/dotfiles/main/.config/copyparty/install.bat -o %tmp%\install.bat && %tmp%\install.bat
```

### install.bat
Installs and update copyparty and cloudflared

### cloudparty.bat
Runs copyparty with cloudflared tunnel

### party.bat
Runs copyparty with the [default config](./copyparty.conf)

### party.py
Runs copyparty from any directory

## Manage dotfiles with GNU Stow
```bash
$ git clone https://github.com/mau-mauricelim/dotfiles.git
$ cd dotfiles
$ stow --adopt . -nv # Simulation mode
$ stow --adopt .
$ git restore . # OR git reset --hard
```
